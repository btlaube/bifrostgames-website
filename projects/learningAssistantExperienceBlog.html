Learning Assistant Program (LAP) Training Workshop

August 19, 2023

Tags: "Key Events"

Before the semester starts the Learning Assitant Program hosts a Traning Workshop that is required for all Learning Assistants and Course Leaders, and is run by the Senior Leaders. The goal of the workshop is to introduce new LAs to the way the program works, what they will be doing, etc. and offers a reminder to the returning LAs and CLs.

This is my 3rd semester with program but my first semester as a Course Leader, so most of the information was review but some was new and helpful. I took careful note of changes within the program since last year in case my lesson plans need to be updated.

The workshop was also a great time to reconnect with fellow LAs after the summer off.


As I mentioned, this is my first semester as a Course Leader. What does that mean? Well, a Learning Assistant has to hold office hours, run weekly lab sections, and grade assignments; there are about 5 LAs for this course. A Course Leader does everything a LA does with the addition of weekly meeting course intructors and senior leaders, bridging the communication gap between instructor/program and LAs, assigning grading, and quality checking grading.

Answering A Request from A Learning Assistant

September 4th, 2023

Tags: "Issues Resolved", "Lessons and Skills"

Today I got a request from a Learning Assistant. They asked to see a solution to the coding assignment to better help students. I thought it would idea for the Learning Assistants to have access for a solution, especially one that the instructor had written.

A solution could be useful for the LAs because they could use it to learn more about the students or prepare better answers to students' questions.

At a meeting with the instructors, I brought up the LA's question, and was able to convince the instructors that it was a good idea.

I got solution code and gave it to the other LAs and made sure to explain the best way to use the code to assist in their teaching. I made sure to highlight a few key things:

  • Don't show the code (the code should be used as reference only).
  • This solution is not the only solution. If a student has another solution, then help them with their solution.
  • Stay vague and try to lead the student to the answer as opposed to telling them outright.

Getting IDEs Installed

September 6th, 2023

Tags: "Issues Resolved"

This week we had our first assignment that required students to run python code locally from their personal machines. Up until now they most students had been writing code using vim on the CSE server and all students had been running their python code on the CSE server.

This assignment required the use of a library not on the CSE server, so, during lab this week, I went through the process of getting students set up with their first Integrated Development Environment (IDE).

People often forget about the set up process that is required before you can start writing code. There are often any number of unexpected errors. After three semesters of going through the same process, I still encounter new errors. However, I have gained plenty of experience helping programming beginners through this process.

Going through this process has developed my ability to proactively react to novel issues and quickly create solutions. Such as skill is appliable in any work environment.

Preparing A Short Lesson on 2D Arrays

September 13th, 2023

Tags: "Issue Resolved"

Navigating the complexities of 2D arrays in an introductory computer science course can be daunting for beginners. Recognizing the need for a streamlined solution, I, as a Learning Assistant, implemented a visual approach to demystify this fundamental programming tool.

In response to recurring student struggles, I utilized the whiteboard to visually represent 2D arrays as grids â€“ a list of lists of elements. Connecting this new concept with their existing understanding of lists, I supplemented the visualizations with concise code examples. By demonstrating how to access elements using two indices, I clarified the grid-like structure, where indices correspond to row and column coordinates.

This visual and practical strategy proved remarkably efficient in helping students comprehend the nuances of 2D arrays. By seamlessly integrating theory with hands-on examples, the approach provides a concise and effective foundation, empowering students to confidently tackle programming challenges.
